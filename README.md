# MINT64 OS란? #
MINT64 OS는 x86 호환 64비트 멀티코어 프로세서에서 동작하는 OS를 만드는 것이 목표입니다.

# MINT64 OS의 기능 #
멀티태스킹과 멀티스레딩, 간단한 파일 시스템, GUI 시스템을 갖추고 있습니다.
